# Okul — Static Scaffold

Это минимальная статическая структура проекта.

Структура:

- `index.html` — главная страница (шаблон).
- `css/style.css` — базовые стили.
- `js/main.js` — базовый JS.
- `assets/` — для изображений и шрифтов.

Как запустить:

Откройте `index.html` в браузере или используйте простой статический сервер, например:

```bash
python3 -m http.server 8000
# затем откройте http://localhost:8000
```
